### My portfolio
### How to start:

1. clone repository: *git clone https://github.com/BrBrov/Portfolio.git*
2. install dependencies: *npm install*
3. start local app:
   - in dev mode: *npm run dev*
   - build: *npm run build* and start preview: *npm run preview*


Link for deploying on Vercel [https://portfolio-brbrov.vercel.app/](https://portfolio-brbrov.vercel.app/)
